# User Management App (React)

React internship challenge app:
- Fetch users from JSONPlaceholder
- Search (name/email)
- Sorting (asc/desc)
- User details page (/users/:id)
- Add new user (local only, with validation, inserted at top)
- Update + Delete using Redux Toolkit

## Tech
- React + Vite
- React Router
- Redux Toolkit
